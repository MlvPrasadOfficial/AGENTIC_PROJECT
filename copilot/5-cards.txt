CARD SYSTEM OVERVIEW
=====================
The application uses a sophisticated card-based layout system with 4 main card types:
1. Upload Card (Left Column)
2. Chat Card (Left Column)
3. Agent Workflow Card (Right Column)
4. Visualization Dashboard Card (Bottom Panel)

CARD ARCHITECTURE MATRIX
=========================
| Card Type | Location | Size | Responsive Behavior | Content Type | Interaction Level |
|-----------|----------|------|---------------------|--------------|-------------------|
| Upload | Left Column | 40% width, 45% height | Adaptive | File operations | High |
| Chat | Left Column | 40% width, 55% height | Adaptive | Text interface | High |
| Agent Workflow | Right Column | 60% width, 100% height | Adaptive | Status display | Medium |
| Visualization | Bottom Panel | 100% width, 40% height | Responsive | Charts/graphs | High |
