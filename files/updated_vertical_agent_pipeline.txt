## AGENT DETAILS

1. **File Upload Agent**
   - Handles file uploads from the UI
   - Validates file formats and initial parsing
   - Displays upload progress and metadata

2. **Data Profile Agent**
   - Analyzes data quality and structure
   - Identifies issues like nulls and outliers
   - Generates data profile statistics

3. **Planning Agent**
   - Parses user queries
   - Determines which agent to activate next
   - Creates execution plan for the workflow

4. **Insight Agent**
   - Generates text insights from data
   - Uses LLM for analysis and explanations
   - Provides narrative summaries

5. **Viz Agent**
   - Creates data visualizations
   - Configures D3.js charts
   - Selects appropriate chart types

6. **Critique Agent**
   - Reviews outputs for quality
   - Identifies potential issues
   - Suggests improvements

7. **Debate Agent**
   - Provides multiple perspectives
   - Examines alternative interpretations
   - Reaches balanced conclusions

8. **Report Agent**
   - Assembles final outputs
   - Generates downloadable reports
   - Formats in PDF, DOCX, HTML
